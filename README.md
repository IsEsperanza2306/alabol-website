# Alabol Car Broker - Website

Sitio web de Alabol Car Broker alojado en Netlify.

## Estructura
```
/                    → Redirige a /links/
/links/              → Página de links (link-in-bio)
/vip/                → Formulario Lista VIP
/precalificacion/    → Formulario Pre-Calificación
/oportunidades/      → Portal de Oportunidades de Subasta
```

## Cómo actualizar las oportunidades
Edita el archivo `oportunidades/oportunidades-data.js` y haz push al repositorio.
